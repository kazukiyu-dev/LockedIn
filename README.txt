LOCKIN TENNIS — GITHUB PAGES FLAT UPLOAD

This version is designed for uploading every file directly into the repository root.
There is no assets folder.

Your GitHub repository should show these files at the top level:
- index.html
- styles.css
- script.js
- success.html
- favicon.svg
- lockin-tennis-advertisement.jpg
- lockin-tennis-advertisement.webp
- certificate-preview.webp
- louis-jeff-aatc-certificate.pdf
- README.txt

Delete any file named "download". It is not part of the website.

GitHub Pages settings:
- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

After uploading, wait for the Pages deployment to finish, then hard-refresh the site with Ctrl+F5.
