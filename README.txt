LOCKIN TENNIS WEBSITE — GITHUB PAGES READY
===========================================

IMPORTANT: Upload the CONTENTS of this folder directly into the repository root.
The repository's main page must show these items at the top level:

- index.html
- styles.css
- script.js
- success.html
- assets/ folder
- .nojekyll

Do NOT upload a containing folder such as "lockin-tennis-website" into the repository.
If index.html is inside a subfolder, GitHub Pages will not use it for:
https://YOUR-USERNAME.github.io/LockedIn/

GITHUB PAGES SETUP
1. Open the LockedIn repository.
2. Delete any empty or old index.html at the repository root.
3. Upload every item listed above directly to the repository root.
4. Open Settings > Pages.
5. Under Build and deployment, select "Deploy from a branch".
6. Select branch "main" and folder "/ (root)", then Save.
7. Wait 1–3 minutes, then hard-refresh the website with Ctrl+F5.
